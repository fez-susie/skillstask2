Inspect Mode

Using Inspect Mode in your browser will allow you to see the HTML code behind all the elements. You can also access the CSS style sheets and JavaScript programs which are relevant to the page you on. Setting the dimensions at the top to different devices will allow you to see how the website behaves on different platforms. Using responsive dimensions will allow you to adjust the webpage’s size as you please.

To access Inspect Mode, right click and press ‘Inspect.’ This feature should be available for most browsers including (but not limited too):

  • Google Chrome
  • Microsoft Edge
  • FireFox
  • Opera
  • Apple’s Safari
